## 말투 매트릭스 작성법

말투 매트릭스는 호칭 일관성 오류를 완전히 제거한다.

### 전체 매트릭스 (모든 캐릭터 × 모든 캐릭터)

`canon/characters/_honorific-matrix.md` 파일을 따로 만들어 관리한다.

```markdown
# 호칭 & 경어 매트릭스

## 읽는 법: 행(話者) → 열(聽者)

|       | 지민 | 철수 | 영희 |
|-------|-----|-----|-----|
| **지민** | - | 야, 반말 | 선배님, 존댓말 |
| **철수** | 지민아, 반말 | - | 영희씨, 존댓말 |
| **영희** | 지민이, 반말 | 철수씨, 존댓말 | - |

## 변화 규칙
- 변화는 반드시 사건 선행 (감정 변화만으로 호칭 변경 불가)
- 변화 이력은 relationship-log.md에 기록
```

## 정보 보유 현황 작성법

"누가 무엇을 알고 있는가"는 드라마 텐션의 핵심이다.
AI가 이를 모르면 캐릭터가 알아야 할 것을 모르거나, 몰라야 할 것을 아는 오류를 낸다.

### 비대칭 정보 기록 예시

```markdown
# 정보 비대칭 현황 (knowledge-map.md 요약)

## 핵심 비밀과 공개 범위

| 비밀 내용 | 아는 사람 | 모르는 사람 | 공개 시점 예상 |
|---------|---------|---------|-----------|
| 지민의 진짜 출생 | 작가(독자) | 지민, 철수, 영희 전부 | 10화 |
| 철수가 배신자 | 작가, 영희 | 지민, 독자 일부 | 7화 |
| 조직의 실체 | 작가, 배후 세력 | 모든 주인공들 | 미정 |
```

## 비주얼 노벨용 스프라이트 설정

비주얼 노벨은 캐릭터 표정/의상 상태를 추가로 관리해야 한다.

`canon/sprites/[이름]-sprites.md`:

```markdown
# [캐릭터 이름] 스프라이트 목록

## 표정 세트
| 코드 | 표정 | 사용 씬 조건 |
|-----|-----|-----------|
| normal | 기본 | 일상 대화 |
| happy | 웃음 | 긍정적 사건 |
| sad | 슬픔 | 이별, 상실 |
| angry | 분노 | 갈등 씬 |
| surprised | 놀람 | 반전 씬 |
| blush | 수줍음 | 로맨틱 씬 |
| serious | 진지 | 결전 씬 |

## 의상 세트
| 코드 | 의상 | 등장 아크 |
|-----|-----|---------|
| casual | 기본 사복 | 1아크 |
| uniform | 교복 | 1-2아크 |
| formal | 정장 | 특별 이벤트 |

## 스크립트 사용 예 (Ren'Py)
\`\`\`renpy
show 지민 happy at left
"지민" "드디어 해냈어!"
show 지민 serious
"지민" "하지만 아직 끝나지 않았어."
\`\`\`
```

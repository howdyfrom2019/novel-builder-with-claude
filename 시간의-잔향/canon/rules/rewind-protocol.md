# REWIND 프로토콜 — 기술 세부 설계

> 작가용 기술 참조 문서. 집필 시 이 파일의 규칙을 위반하면 안 됨.

---

## REWIND란 무엇인가

**R**etrocausal **E**ngineered **W**aveform **I**mprint **N**eural **D**evice

직역: 역인과 설계 파형 각인 신경 장치

**개발 목적:** 인간의 기억 패턴을 압축 신호로 변환해 역시간 방향으로 전송함으로써, 동일 개체의 과거 신경망에 각인시키는 기술.

**현실 개발 배경 (작중 설정):**
- 2013년, 국립 양자신경과학연구소 (NURI) 설립과 함께 이론 정립 착수
- 개발 착수 동기: 외상 후 스트레스 장애(PTSD) 치료 목적 — 외상 기억의 "재각인"을 통해 트라우마를 처리
- 비의도적 발견: 실험 중 역방향 신호 전송 가능성 확인
- 비밀 프로젝트로 전환: 군사 및 정보기관 관심 유치

## 도약 메커니즘 (단계별)

### Phase 1: 스캔 (Scan)
```
소요 시간: 약 3분
대상: 도약자의 뇌 전체 신경 연결망
방식: 비침습적 고해상도 신경 매핑
출력: Raw Neural Pattern (RNP) — 약 240TB 상당의 정보량
```

### Phase 2: 압축 (Compression)
```
소요 시간: 약 12분
알고리즘: 의미론적 신경 압축 (Semantic Neural Compression, SNC)
- 단기기억 vs 장기기억 분류
- 감정 태그 보존 우선
- 감각 기억 (시청각 등) 은 저해상도로 압축
출력: Compressed Memory Signal (CMS) — 약 0.3TB
손실: 약 3~8% (경험 미숙 운용자) / 1~2% (숙련 운용자)
```

### Phase 3: 채널 개통 (Channel Opening)
```
소요 시간: 약 8분
원리: 역인과 양자 채널 (Retrocausal Quantum Channel)
- 제로포인트 에너지장을 이용한 채널 개통
- 목표 시점의 오차 범위: ±90초
- 목표 좌표: 도약자의 과거 신경 패턴 위치 (고유 신경 지문으로 특정)
한계: 72시간 이상 역행 시 채널 안정성 급감
```

### Phase 4: 전송 및 각인 (Transmission & Imprint)
```
소요 시간: 약 0.3초 (전송 자체는 순간적)
수신 주체: 과거 시점의 도약자 본인 뇌
각인 방식: 기존 신경 패턴 위에 덮어쓰기 (Overwrite)
부작용: 수신 직후 30~90초간 의식 혼미 상태 (잔향)
```

## 도약자 자격 조건

모든 인간이 기억 도약이 가능하지 않다.

**Leap Receptor (도약 수용체):**
뇌의 해마-시상 연결부에 존재하는 특수 신경 수용체. 역인과 신호를 수신할 수 있는 구조.

```
인구 중 수용체 활성화 비율: 약 0.003%
수용체 잠재 보유자 (미활성): 약 0.12%
```

**활성화 조건:**
- 선천적 활성: 극소수 (대부분 무의식적 "직관"으로 발현)
- 후천적 활성: REWIND 신호에 노출 시 수용체 자극으로 활성화 가능

**잔향 감지자:**
수용체가 잠재 상태인 사람은 도약자의 "잔향"을 감지하는 능력을 보임. 이것이 수용체 활성화의 전조 증상.

## REWIND 장치의 물리적 형태

### 현재 버전 (Mark-3):
- 크기: 냉장고 크기의 메인 유닛 + 헬멧형 수신 모듈
- 전력 소비: 전력망 직접 연결 필요 (일반 건물 수용 불가)
- 위치: NURI 지하 2층 실험실

### 이전 버전 (Mark-1, Mark-2):
- 개발 초기, 정확도 낮음
- Mark-1 자기 실험으로 이석호가 수용체를 후천 활성화 (스토리 배경)

### 미래 버전 (이야기 후반 등장):
- 휴대 가능한 소형화 버전 (이야기 내 갈등의 핵심)
- 소형화 = 더 많은 사람이 도약할 수 있음 = 더 많은 세계선이 "어두워짐"

## 도약 기록 코드

REWIND 시스템이 도약 이벤트를 기록하는 고유 코드.

```
형식: [도약자 신경 지문 ID]-[출발 시점]-[도착 시점]-[세계선 분기 해시]
예시: JY-2026.09.14.22:17-2026.09.12.08:00-WL#4A9F
```

이 기록은 시스템에만 남으며 도약자 본인은 접근 불가 (보안 등급 최상위).

## REWIND 사용의 윤리 문제 (작중 쟁점)

1. **동의 없는 도약**: 도약 후 과거의 자신이 경험하는 기억 덮어쓰기는 과거의 자신의 동의가 없음
2. **이면 세계의 소멸**: 도약으로 인해 잠재 상태가 되는 세계선의 구성원들의 권리
3. **국가 독점 문제**: REWIND가 특정 기관에만 허용될 경우의 권력 불균형
4. **기억 진실성**: 도약자의 기억이 "조작된" 것인지 "경험된" 것인지의 법적 지위

이 쟁점들은 작중 캐릭터들이 직접 논쟁한다. 명확한 답은 제시하지 않는다.

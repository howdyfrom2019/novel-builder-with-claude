# 세계관 규칙 — 시간의 잔향 (殘響)

> 이 파일은 모든 집필의 물리적 헌법이다. 위반 불가.

---

## 철학적 전제 (물리 법칙 이전에 오는 것)

이 세계관의 모든 물리 규칙은 하나의 철학적 명제로부터 파생된다.

**"미래를 모르는 것은 결핍이 아니다. 그것이 인간이 살 수 있는 조건이다."**

기억 도약(REWIND)은 과학적 돌파구가 아니다.
그것은 인간이 살 수 있는 조건으로부터의 이탈이다.

- 도약 이전의 인간 → 불확실성 속에서 선택. 그 자체가 숭고하다.
- 도약 이후의 존재 → 이미 본 것을 향해 걷는다. 그것이 저주다.
- 누적 손상과 기억 소실 → 저주의 물리적 표출. 너무 많이 알려 한 대가.

**어트랙터는 우주가 말하는 것이다: "어떤 것은 알도록 허락되지 않았다."**
도약으로 어트랙터를 이탈하지 못하는 이유는 물리적 제약 때문이기도 하지만,
더 근본적으로는 알고 나서 바꾸려는 행위 자체가 이미 허용 범위를 넘었기 때문이다.

## 세계관 기본 전제

### 우주의 구조: 다층 세계선 체계

우주는 단일하지 않다. 양자적 결정이 일어나는 순간마다 세계선(世界線)이 분기한다. 이것은 에버렛의 다세계 해석(Many-Worlds Interpretation)에 기반한다.

그러나 우리 세계관에는 하나의 결정적 추가 조건이 있다.

**관측 붕괴 법칙 (Observation Collapse Law):**
> 분기된 세계선은 "관측자 (Observer)"가 존재하는 한에서만 실재(實在)한다.
> 관측자가 없는 세계선은 파동함수가 붕괴되지 않아 중첩 상태(叠加態)로 전락한다.
> 살아있는 우주 = 관측되는 우주. 잠재 우주 = 관측되지 않는 우주.

```
[관측 있음]  세계선 A ─────────────── 실재 (Real)
             ↓ 분기
[관측 없음]  세계선 B ┄┄┄┄┄┄┄┄┄┄┄┄┄ 잠재 (Potential)
```

잠재 상태의 세계선은 소멸하지 않는다. 다만 실재하지 않을 뿐이다. 새로운 관측자가 진입하면 다시 실재로 붕괴될 수 있다.

**잠재 세계선의 구성원:** 잠재 세계선에 남겨진 사람들은 의식을 잃지 않는다. 그들은 존재하지만, 그 존재가 실재 세계선의 물리 법칙에 의해 관측되지 않는다. 이것이 도약의 윤리적 핵심이다 — 도약자는 세계를 버릴 때, 그 세계의 사람들을 함께 버린다.

## 핵심 이론 3종

### 1. 타임 패러독스 이론 (Time Paradox Theory)

미래의 자신에게 영향을 끼칠 행동을 과거에 수행하면, 현재 세계선의 인과 구조가 모순에 빠진다.

**발현 메커니즘:**
- 과거의 행동 A → 미래의 자신 B가 존재하지 않거나 달라짐
- B가 달라지면 과거로 보낸 기억 신호의 내용도 달라졌어야 함
- 이미 보낸 신호와 "보냈어야 할" 신호가 충돌 → 세계선 재구축

**재구축 (Reconstruction):**
세계선이 인과 모순을 해소하기 위해 사건 배열을 재편하는 현상. 재구축은 미시적(사건 한두 개)에서 거시적(아크 전체)까지 범위가 다양하다.

**중요:** 재구축은 기억 도약자의 의지와 무관하게 발생한다. 예방할 수 없다.

### 2. 에너지 증가 패러독스 (Energy Increase Paradox)

**왜 물리적 시간여행은 불가능한가:**

열역학 제1법칙에 의해 폐쇄 우주 내의 총 에너지는 보존된다.
인간 + 장치가 과거로 이동한다 = 과거 시점 우주에 질량이 추가됨 = 우주 총 에너지 증가 = 물리 법칙 위반.

```
[현재 우주 에너지] = E
[과거로 이동 후 과거 우주 에너지] = E + (탑승자 질량 × c²)
→ 모순. 불가능.
```

**해결책: 기억 도약 (Memory Leap)**

정보(Information)는 질량이 없다.
뇌의 신경 패턴 — 기억, 경험, 판단의 정보 구조 — 을 순수 정보 신호로 압축하면 에너지 보존을 위반하지 않는다.

```
[전송물] 신경 패턴 압축 신호 (질량 = 0, 정보량 = 有)
[경로]   역시간 방향 양자 채널 (Retrocausal Quantum Channel)
[수신]   과거 시점 동일 개체의 뇌 (수신 매체, 질량 보존)
[결과]   과거의 몸, 미래의 기억
```

**기억 도약의 물리적 작동:**
1. 대상 뇌의 신경 연결망 전체를 스캔 (비파괴적)
2. 패턴을 역시간 양자 신호로 인코딩
3. REWIND 장치가 역인과 채널 개통
4. 과거 시점 뇌의 수용체(Leap Receptor)에 신호 수신
5. 신호 덮어쓰기 — 과거의 기억 위에 미래의 기억이 중첩

**절대 원칙:** 이 세계관에서 타임머신은 존재하지 않는다. 물리적 역행도 존재하지 않는다. 오직 기억 신호의 역행만 가능하다.

### 3. 확률 밀도 수렴 이론 — 어트랙터 (Attractor Field)

**[개정판 — 구 "결과 회귀 고립 이론"을 대체]**

REWIND는 역인과 채널을 물리적으로 개통한다. 채널 개통 과정은 국소 시공간의 양자 확률 장(quantum probability field)을 교란한다. 이 교란은 특정 원인을 고정시키지 않는다. 대신, **특정 영역 내 사건의 확률 밀도 분포를 왜곡**한다.

```
REWIND 채널 교란 효과:
정상 시공간 → 사건 확률이 무작위 분포
교란 시공간 → 특정 종류의 사건(외상적 사건)의 확률 밀도가 집중됨

예시:
- NURI 반경 X 내에서 화재 발생 확률 = 정상의 N배
- 원인이 제거돼도 확률 밀도는 유지
- 새로운 원인이 그 확률 밀도를 채움
```

**어트랙터의 올바른 정의:**
- 어트랙터 = 특정 종류의 사건이 높은 확률로 발생하도록 수렴된 확률 밀도 집중 영역
- "화재 원인이 고정된 것"이 아니라 "화재 확률이 구조적으로 높아진 것"
- 원인 사슬의 끝을 자르면 새 원인이 생긴다 — 확률 밀도가 채워지는 방식으로

**핵심 결론:**
어트랙터는 "원인의 고정"이 아니라 "확률장의 변형"이다.
따라서 어트랙터를 해소하는 방법은 확률장의 교란을 멈추는 것, 즉 REWIND 사용을 중단하는 것이다.

```
REWIND 사용 지속 → 교란 누적 → 확률 밀도 집중 강화 → 어트랙터 심화
REWIND 사용 중단 → 교란 소멸 → 확률 밀도 정상화 → 어트랙터 해소
```

**예: NURI 화재 어트랙터**
- 아크 2의 연구소 화재는 특정 원인(배선, 가스)이 고정된 게 아님
- NURI에서 REWIND를 반복 사용한 결과 국소 확률장이 교란되어 화재 확률 밀도가 집중됨
- 재윤이 배선을 고쳐도 가스 누출이 발생하고, 가스를 막아도 또 다른 원인이 나타남
- 해법: REWIND 자체를 멈추는 것 (도약 중단)

**어트랙터 강도:**
- 도약 횟수에 비례해 확률장 교란이 누적됨
- 초기(1~5회): 교란 약함. 확률 밀도 집중 미미.
- 중기(6~20회): 교란 중간. 특이 사건 발생 빈도 증가 가시적.
- 후기(20회+): 교란 강함. 어트랙터가 구조화됨. 도약자 혼자 해소 불가 수준.

## 기억 도약의 한계와 부작용

### 용량 한계 (Capacity Limit)
뇌 신경 패턴의 압축률에 물리적 한계가 있다.
- 최대 역행 가능 시간: 현재 기술 기준 72시간
- 72시간 이상의 기억을 보내면 압축 과정에서 데이터 손실 발생
- 손실된 기억은 복구 불가

### 역방향 잡음 (Backward Noise)
도약 직후 일부 기억이 흐릿하게 변형되거나 순서가 뒤섞이는 현상.
- 도약자는 도약 후 수십 분간 혼란 상태

### 잔향 (Resonance) — 개정판

**잔향의 물리적 정의:**
도약자가 세계선을 이탈할 때, 이탈된 세계선은 관측자를 잃고 잠재 상태로 향한다. 잠재화 직전, 해당 세계선에서 도약자와 강하게 감정적으로 얽혀 있었던 인물의 기억 패턴이 역인과 채널의 잔류 신호를 타고 활성 세계선으로 흘러든다. 이것이 잔향이다.

**잔향이 발생하는 조건:**
1. 도약자가 세계선을 이탈 (잠재화 시작)
2. 잠재화 직전 → 도약자와 강하게 감정 얽힌 인물의 기억 패턴이 "방전"됨
3. 방전된 패턴이 역인과 채널 잔류 신호에 올라타 활성 세계선으로 전달
4. 활성 세계선의 수신자가 그것을 받아들임

**왜 정아름이 잔향을 받는가:**
- 아름은 잠재 수용체 보유자 — 역인과 신호에 수동적으로 공명하는 신경 구조가 있음
- 매 도약마다 잠재화된 세계선의 "아름"들이 방전한 기억 파편이 현재 아름에게 흘러들어옴
- 도약 횟수가 쌓일수록 잔향 신호의 강도와 빈도가 올라감
- 아름에게 들리는 것은 자기 자신의 다른 버전이 남긴 흔적이다

**잔향의 내용:**
- 완전한 기억이 아닌 파편 (감각적 단편 또는 짧은 음성)
- 내용은 항상 높은 감정적 강도를 가짐 (감정 태그가 잔향에 잘 살아남는 이유)
- 아름이 받는 잔향 내용의 전형: 위기 상황에서 재윤을 찾는 자신의 목소리

**유하린의 잔향:**
- 하린은 도약자이기도 하므로 잔향을 수신뿐 아니라 능동적으로 생성할 수도 있음
- 하린이 아름 앞에서 가끔 감정이 새어나오는 이유 — 다른 세계선의 자신(= 아름)이 방전한 기억이 하린 자신에게도 역류하기 때문

### 누적 손상 (Cumulative Damage)
- 도약 1~5회: 두통, 일시적 기억 혼란
- 도약 6~15회: 감정 반응 둔화, 특정 기억 영구 손실
- 도약 16~30회: 인격 통합 장애, 다른 세계선 기억이 현재 기억에 침투
- 도약 30회 이상: 비가역적 신경 손상. 특정 감각 기능 소실. 현재 세계선과의 동기화 실패
- **30회 이상 도약자는 불완전 기억 보존자다.** 기억의 일부가 세계선 간 이동에서 손실된다. 기록 시스템 없이는 자신이 경험한 것조차 신뢰할 수 없게 된다.

### 자연 발화 수신 (Natural Receptor Activation)

선천적 수용체가 활성 상태인 개인은 REWIND 장치 없이도 역인과 신호를 수신할 수 있다.

**발생 조건:**
- REWIND 실험이 발생시키는 역인과 채널 "잔류 신호"를 수신 (거리 무관)
- 또는 미래의 자신이 도약을 수행할 경우, 그 신호가 과거의 수용체에 역방향 도달

**수신 특성:**
- 내용: 완전한 기억이 아닌 단편 (시각·청각·촉각·후각의 파편)
- 지속 시간: 3~15초
- 강도: 기계 전송의 5~10% — 흐릿하고 단편적

**NURI가 이것을 탐지하는 방법:**
REWIND 실험 중 발생하는 채널 신호의 "반향(echo)"을 역방향으로 추적하면
수신 이벤트가 발생한 개인의 위치를 특정할 수 있다.
이것이 NURI가 이재윤을 발견한 방법이다.

### 관측자 역설 (Observer Paradox)
도약자가 현재 세계선을 떠나면, 그 세계선에서 도약자라는 관측자가 사라진다.
- 남은 사람들이 관측자 역할을 이어받으면 세계선은 생존
- 도약자가 유일한 관측자였다면 세계선은 잠재 상태로 전락
- **윤리적 무게:** 도약 = 하나의 세계를 사실상 버리는 행위

## 절대 규칙 (위반 불가)

1. **물리적 역행 불가**: 어떤 수단으로도 질량이 있는 물체는 과거로 보낼 수 없다
2. **기억 도약 역전 불가**: 한번 덮어써진 과거 기억은 복구할 수 없다
3. **미래 기억 선행 전송 불가**: 아직 일어나지 않은 미래의 기억을 전송할 수 없다
4. **동일 시점 중복 도약 불가**: 같은 시점에 두 번 이상의 도약 신호를 수신하면 신경 과부하로 즉시 사망
5. **관측자 없는 세계선은 잠재 상태**: 소멸하지는 않으나 구성원들의 실재 여부가 불확실해짐
6. **어트랙터는 존재한다**: 세계선은 무한히 자유롭지 않다. REWIND 사용으로 교란된 확률 밀도 집중은 반드시 어트랙터를 형성한다
7. **어트랙터의 해소는 교란의 중단으로만 가능하다**: 원인 사슬 조작만으로는 어트랙터 이탈 불가

# 관계 변화 로그 — 시간의 잔향

| 화수 | 관계 | 변화 내용 | 근거 사건 |
|-----|-----|---------|---------|
| 초기 | 재윤↔아름 | 동료. 점심을 자주 같이 먹음. 약간의 호감 | 6개월간의 일상 |
| 초기 | 재윤↔이석호 | 피실험자-연구책임자. 재윤은 이석호의 두 상태(면담실/3층)를 다른 인물처럼 인식 | 1~4화 |
| 초기 | 재윤↔하린 | 동료-선배. 설명할 수 없는 친근감 | 재윤 일방적 감각 |
| 초기 | 아름↔하린 | 후배-선배. 친함 | 연구소 내 관계 |
| 초기 | 이석호↔하린 | 상급자-조교 (표면상) | 공식 관계. 이석호는 하린의 정체를 완전히 파악 못 함 |
| 5화 | 재윤↔이석호 | 피실험자→프로젝트 참여자. B2 접근 허가. 정보 공유 시작 | 재윤이 참여 결정. 이석호가 JY 코드, [UNKNOWN] 공개 |
| 5화 | 재윤↔아름 | 세 번째 만남. 낯섦 소멸, 익숙함 단계. 아름의 관찰이 구체화 | 아름이 손 움직임·습관 변화를 직접 언급 |
| 5화 | 재윤↔진서 | 친구. 재윤의 변화를 감지하지만 묻지 않는 관계 유지 | 진서 "너 요즘 뭔가 다르다" — 재윤 무응답 — 불편하지 않은 침묵 |
| 6화 | 재윤↔아름 | NURI 밖 첫 접촉 (신림동 원룸). 데이터 공유 → 개인적 대화 개방 | 아름이 재윤의 집 방문. 밤 수신 고백. "이석호가 뭔가를 알고 있다" 인식 공유 |
| 7화 | 재윤↔이석호 | B2에서 최초 단독 대화. 이석호 = REWIND 창시자·도약자 확정 | 이석호가 직접 인정. "축적돼요." 위치가 바뀜 (재윤이 서고 이석호가 앉음) |
| 7화 | 재윤↔아름 | 이석호 정보를 상호 공유하는 관계로 전환 | 이석호 왼손-하린 왼손 대칭 동시 인식. "소장님 왼손" / "하린 씨 왼손" 교환 |
| 8화 | 재윤↔진서 | 재윤이 "아직 말 못 해" 직접 언급. 진서가 수용 | 진서는 더 묻지 않음. NURI 내부 정보(이석호 야간 B2 출입) 제공 |
| 8화 | 재윤↔아름 | NURI 밖 두 번째 만남 (카페). 독립 분석 공유. 이석호에 대한 경계 공유 | 아름이 "도약을 한 적 있어요?" 직접 질문. 아름이 데이터를 연구소 시스템에 올리지 않기로 결정. "소장님이 알고 있는 것과 우리가 알고 있는 게 같은 방향인지 모르겠어요" |
| 9화 | 재윤↔이석호 | 단독 B2 세션. 이석호가 수용체 능력을 부분 공개하되 정보 범위를 통제 | 헬멧 없는 "확인" 세션. 수용체=채널 개통 장치 공개. "고정점" 발언 (미설명). "그것도 맞아요" = 다른 이유 미공개. 혼탁 전환 "재현아, 가지 마" → 재윤이 "재현이 누구예요?" → 이석호 최초 시선 회피 |
| 9화 | 재윤↔하린 | 로비 조우. 하린이 이석호에 대한 경고를 재윤에게 직접 전달 | "소장님이 보여주는 것만 보면 안 돼요." "고정점" 단어를 B2 밖에서 예측. "처음엔 다 그래요" 도약 연관성 직접 질문 → 회피. 재윤이 하린의 뒷모습과 비전 속 여자의 겹침을 감각적으로 인식 |
| 9화 | 재윤↔아름 | 텍스트 공유. 아름에게도 "혼자 오세요" 연락이 왔음을 확인 | 이석호의 분리 전략을 양측이 동시 인식. 재윤이 "만나서 얘기할게요"로 대면 약속 |

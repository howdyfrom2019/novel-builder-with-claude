# 캐릭터 지식 지도 — 시간의 잔향 (아크 시작 기준)

## 이재윤 (주인공) — Main WL, 9.10 (1화 기준 / 첫 도약 이전)

- **아는 것 (1화 기준)**: 오후 두시 십칠 분 골목에서 약 10초간 이상한 감각 경험. 흰 복도, 형광등, 여자의 등, 발소리. 설명 못함. NURI 임상 공고에 지원함 (동기: 돈). 공고를 진서로부터 받음.
- **모르는 것**: REWIND의 존재. 도약의 존재. 자신의 수용체. 자연 수신이 무엇인지. NURI가 자신을 타겟팅했다는 것. 아름의 존재 (이름만 독자에게 예고됨).

> **2화 이후 갱신 (9.14 기준 / Main WL / 1차 방문 후)**

> - **아는 것**: NURI 건물 내부 구조(로비, 2층 세션룸, 카페테리아, 4층 소장실). 복도가 자연 수신 장면과 유사함 (결론 보류). 정아름 (세션 안내 담당). 유하린 (조교, 뒷모습이 수신 여자와 유사하나 머리 길이 불일치). 이석호 소장의 두 가지 상태 목격 (같은 인물인지 확신 못 함). "재현"이라고 불린 것. JY-WL#1A/1B/2A 코드 일부 목격 (자기 이니셜 의심, 판단 유보). 이석호 발언 2건: "바꿀 수 없는 걸…" / "첫 번째가 가장 선명해요".
> - **모르는 것**: REWIND의 존재. 도약의 존재. 자신의 수용체. 자연 수신이 무엇인지. NURI가 자신을 타겟팅했다는 것. JY 코드의 의미. 이석호의 "재현" 발언의 의미. 하린의 정체.

> **3화 이후 갱신 (9.12 기준 / WL#1 / 첫 도약 직후)**
>
> - **아는 것**: 2화까지의 모든 기억(9.14 방문) + 의식이 9.12로 되돌아갔다는 사실. 9.14의 기억이 선명하나 해상도 비균일(일부 흐릿). 9.12 밤 청각 파편 수신 — "재윤 씨, 지금—" (아름 목소리와 유사). 패턴 인식: 9.10 시각수신 → 9.14 의식전환 → 9.12 청각수신 (간격 축소 추세).
> - **모르는 것**: 도약이 무엇인지. REWIND. 자신의 수용체. 아름 목소리 파편의 정체. 왜 시간이 되돌아갔는지. 이석호와의 연관. NURI에서 무엇이 발생했는지.

> **4화 이후 갱신 (9.14 기준 / WL#1 / 2차 방문 후)**
>
> - **아는 것**: "도약" 개념 (기억이 시간을 거슬러 올라가는 현상). B2 장비 오작동으로 역인과 채널이 열렸고 그것이 사고 원인. 자신에게 선천적 수용체가 있음. 프로젝트 참여 제안 수신 (보류). 기억과 현실이 80% 이상 일치함을 직접 검증. 유하린이 B2에 대해 알고 있음.
> - **모르는 것**: REWIND 장치의 정체(이름). 도약의 물리적 원리. 역인과 채널이 무엇인지. 이석호가 왜 가끔 다른 사람처럼 보이는지 (현상만 인식, 원인·명칭 모름). 하린의 정체. JY 코드의 의미. 이석호의 단일 통합 계획. 수용체가 무엇을 위한 것인지.

> **5화 이후 갱신 (9.28 기준 / WL#1 / 3차 방문 후)**
>
> - **아는 것**: 4화까지의 모든 지식 + 프로젝트 참여 결정. B2 직접 방문. REWIND 장치의 물리적 형태 (냉장고 크기 유닛 2개, 헬멧, 의자). 장치의 공식 명칭 "신경 패턴 캘리브레이션 장치" (비공식 명칭 존재 인식). JY-WL#1A~#12B 코드 24개 직접 확인 (JY = 도약자 신경 지문 코드). [UNKNOWN] 코드 발견 (날짜·번호 없음, 시스템 자체 생성). "일부는 실현, 일부는 예측 코드" (구분 불가). 장치 앞 바닥 마모 흔적 (이석호 회피 답변). "이 장비를 사용한 사람이 있었다. 지금은 당신." 아름의 이중 피크 패턴 관찰 (일반 참여자에게 없음). 기억 해상도가 비균일하게 변하고 있음 (이석호의 "첫 번째가 가장 선명" 발언이 맞고 있음). 진서로부터 이석호 소문 ("3층 혼잣말, 4층과 사람이 다르다").
> - **모르는 것**: REWIND라는 이름. 도약의 물리적 원리 (역인과 채널의 구체적 메커니즘). JY 코드 중 어느 것이 실현/예측인지. [UNKNOWN]의 정체. 하린의 정체. 이석호가 가끔 다른 사람처럼 보이는 이유 (왔다갔다한다는 감각만 있음. "명료/혼탁"이라는 분류 자체를 모름). 이석호의 단일 통합 계획. 바닥 마모 흔적의 실제 원인. 이전 장비 사용자의 정체. 이중 피크 패턴의 의미.

> **6화 이후 갱신 (10.12 기준 / WL#1 / 블랙아웃 + 야간 긴급 방문)**
>
> - **아는 것**: 5화까지의 모든 지식 + 밤 잔향 수신 빈도 폭증 추세: 10.03→10.05→10.08→10.12 (간격 축소, 길이 증가). 목소리의 절박함 인식 ("찾고 있는 사람의 목소리"). 10.12 낮 주간 수신 최초 발생 — 눈을 뜬 상태에서 시각(흰 복도)+청각("재윤 씨, 지금 거기 있어요?") 파편. 이후 블랙아웃 (2시간 의식 상실). 수용체가 장비 없이도 원격 공명한다는 사실 (B2 장비가 꺼진 상태에서 14:02에 비정상 파형 수신 — 블랙아웃 시각 일치). 아름이 B2 모니터링 데이터를 근거로 재윤의 집에 직접 방문. 아름에게 밤 수신 사실 + "아름 씨 목소리와 비슷" 고백. 수신이 9.12(첫 세션 이전)부터 시작됐음을 아름에게 밝힘. 아름이 원룸에서 오른손→왼쪽 볼 터치 — 하린과 동일 동작임을 선명하게 인식. 이석호가 전화 중 "재현이한테…" 실수 발언 (아름도 들음). 아름도 이석호가 "같은 사람인데 다른 사람처럼 느껴진다"고 인식함을 확인. 이석호 긴급 호출로 밤에 NURI 방문 — 밤의 NURI 분위기 경험. 하린의 왼손에 물컵 떨림 목격 (검지-중지 간격). B2 도착 시 장비가 켜져 있었다.
> - **모르는 것**: REWIND라는 이름. 도약의 물리적 원리. [UNKNOWN]의 정체. 하린의 정체. 이석호의 두 상태 원인 (현상은 점점 확실해지나 원인·명칭 모름). 이석호의 단일 통합 계획. 바닥 마모 흔적의 실제 원인. 이전 장비 사용자의 정체. 이중 피크가 매번 동일한 이유. 수신되는 목소리의 정체 (아름 유사하지만 "현재의 아름은 아닌 것 같다"는 판단). 아름-하린 동작 일치의 이유. 하린 왼손 떨림의 원인.

> **7화 이후 갱신 (10.12 밤 기준 / WL#1 / 야간 B2 긴급 측정 후)**
>
> - **아는 것**: 6화까지의 모든 지식 + REWIND라는 장치 명칭 (이석호가 "리와인드가 이 수준의 수신자를 상정하고 설계된 건 아닙니다"라고 실수 발언 → 이후 "만든 사람은 리와인드라고 불렀다" 직접 설명). B2 긴급 측정: 수용체가 이미 활성 상태여서 기준선 측정 불가. 측정 중 가장 선명한 수신 — [대답해요. 제발—] + 흰 복도 비전 + 짧은 머리 여자가 돌아서는 장면 (얼굴 확인 직전 경고음). 수용체가 장비에 역방향 신호 전송 → 경고음 → 모니터 빨간색. "리와인드를 만든 이후에 이런 반응을 보인 사람은 없었어요. 나를 포함해서." 이석호가 REWIND의 최초 사용자 = 자신임을 인정 ("리와인드를 만들고, 처음으로 사용한 사람이 나예요"). 이석호도 도약을 경험했다는 사실. 바닥 마모 흔적 = 이석호의 발자국으로 확정. 이석호의 "재현" 호칭이 다른 기억의 혼재임을 추론 (이석호가 부정하지 않음). 이석호 왼손 기능 저하 직접 목격 (키보드 오른손만 사용, 헬멧도 오른손으로 씌움). 하린 왼손 기능 완전 저하 직접 목격 (컵 떨어뜨림, 벽에서 미끄러짐, 팔 전체 힘 빠짐). "추워서" 회피 → 아름 "B2는 항온" 반박. 이석호 혼탁 전환 목격: "유진이도 그랬어. 손끝부터." + "재현아" (셋 앞에서). 이석호-하린 왼손 대칭 인식 (같은 쪽, 같은 증상). 아름도 이 대칭을 동시 인식. "도약을 하면 몸에 무슨 일이 일어나요?" → 이석호 "축적돼요."
> - **모르는 것**: 도약의 물리적 원리 (역인과 채널의 구체적 메커니즘). [UNKNOWN]의 정체. 하린의 정체. 이석호의 두 상태 원인 (도약 축적 때문이라는 추론은 형성 중이나 확정 못 함). 이석호의 단일 통합 계획. 이석호가 도약한 횟수. 이석호가 왜 도약했는지. "유진"이 누구인지. 이중 피크 고정의 물리적 원인. 수신 목소리의 정체 (아름 유사, 현재 아름 아님). 아름-하린 동작 일치의 이유. 수용체가 왜 장비에 역방향 신호를 보내는지. 하린 왼손 증상의 원인 (이석호와의 대칭은 인식, 원인 미확인). 잔향 비전 속 짧은 머리 여자의 정체.

## 정아름

- **아는 것 (7화 이후)**: 재윤이 가끔 "달라 보인다"는 감각. 이석호에게 두 가지 상태가 있다는 것 (단, 설명 못 함). 이중 피크 패턴이 세션마다 동일 (마흔 명 중 유일). "수용체에 지속적으로 신호가 도달 중"이라는 자체 분석. B2 장비가 꺼진 상태에서 14:02에 비정상 파형 수신 → 재윤 수용체의 원격 공명 확인. 재윤의 집을 직접 방문 (쓰러진 의자 목격). 재윤이 밤에 목소리 파편을 듣고 있다는 사실 (재윤 직접 고백). 그 목소리가 "자신과 비슷하다"는 것 (재윤 직접 발언). 수신이 9.12부터 시작됐다는 사실 = 첫 세션 이전. 이석호가 전화 중 재윤을 "재현"이라고 부른 것 직접 청취. 이석호가 B2에서도 셋 앞에서 "재현아" 발언 직접 목격. REWIND라는 장치 명칭 (이석호 실수 발언 직접 청취). 측정 중 수용체가 장비에 역방향 신호 전송한 것을 데이터로 확인 ("역방향이에요" 직접 분석). 하린 왼손 기능 저하 직접 목격 (부축). 이석호 왼손 기능 저하 인식 (재윤과 "소장님 왼손" / "하린 씨 왼손" 교환). 이석호가 REWIND를 만들고 직접 사용한 사람이라는 사실 (재윤에게서 들음). 이석호도 도약했다는 사실. "그 말을 할 것 같아요" — 수신 목소리가 자신의 미래일 수 있다는 무의식적 감각.
- **모르는 것**: 도약의 존재 (개념 자체를 아직 설명받지 못함. 이석호가 "도약"을 했다는 사실은 알지만 도약이 무엇인지 정확히 모름). 자신의 미래. 하린의 정체. 이중 피크의 물리적 원인. 목소리가 왜 자신과 비슷한지. 이석호의 단일 통합 계획. "유진"이 누구인지.

## 이석호

- **아는 것**: REWIND 전체. 재윤의 수용체. 단일 통합 계획. 47회 도약 경험. 자신이 아름의 잔향 감지 능력을 제거해야 통합이 안정적임.
- **모르는 것**: 하린의 정체 (완전히). 통합이 또 다른 어트랙터임을 모름. 자신의 계획의 물리적 오류를 아직 인식 못 함.

## 유하린 (WL#9X 아름)

- **아는 것**: 거의 모든 것. 단, 현재 시점에서 직접 말할 수 없음.
- **모르는 것**: 재윤이 이번 사이클에서 올바른 선택을 할지.

## 김진서

- **아는 것**: NURI 전산 시스템 (B1 근무). 재윤이 요즘 "뭔가 다르다" (평소의 취업 고민과 다른 종류의 생각). 이석호에 대한 소문 (3층 혼잣말, 4층 면담 때와 사람이 다르다). NURI 건물 분위기가 "빛이 없다"는 감각.
- **모르는 것**: REWIND. 도약. 모든 것. 자신이 재윤을 NURI로 인도한 트리거 역할이라는 것.

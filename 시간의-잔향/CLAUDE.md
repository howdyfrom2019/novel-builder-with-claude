## 문체 규칙

- **시점**: 3인칭 제한 시점 (이재윤 시점 기본, 필요 시 다른 인물로 전환 가능)
- **문체**: 건조하고 정밀함. 과학 개념은 비유로 설명하되 틀리지 않게
- **화당 목표 분량**: 8,000~12,000자
- **특이사항**:
  - 도약 발생 시: `◆ LEAP #N ◆` 표기 후 시점 전환
  - 잔향 수신 시: 이탤릭체 처리 또는 `[ ]` 괄호 묘사
  - 세계선 코드는 `WL#코드` 형식으로 표기
  - 장면 전환: `* * *`

## 캐릭터 말투 원칙

- **재윤**: 간결하고 단계적. "우선 A, 그다음 B" 식 구조화 말투
- **아름**: 직접적 질문형. 감정과 논리를 동시에 사용
- **이석호 (명료 상태)**: 반문형. 침묵을 자주 사용. 격식체 기본. 감정 표현 거의 없음.
- **이석호 (혼탁 상태)**: 두서없다가 간헐적으로 선명해짐. 재윤을 다른 이름으로 부를 수 있음.
- **하린**: 부드럽고 느림. 유도 질문형. 직접 답하지 않음
- **진서**: 구어체, 단도직입, 감탄사 많음

## 구조 반복 금지 (anti-pattern 규칙)

에피소드가 동일 패턴에 빠지는 것을 금지한다. 집필 전 직전 2~3화의 구조를 반드시 확인하고, 아래 항목 중 3개 이상 겹치면 구조를 변경해야 한다.

**패턴 체크리스트 (3개 이상 연속 반복 시 경고):**
- 시작 장소가 같은가 (예: 매화 NURI 도착으로 시작)
- 중심 이벤트가 같은 유형인가 (예: 매화 세션/측정)
- 정보 전달 방식이 같은가 (예: 매화 이석호가 설명)
- 종료 구조가 같은가 (예: 매화 버스/택시로 귀가 + 내면 정리)
- 등장인물 조합이 같은가 (예: 매화 재윤-아름-이석호-하린 전원 등장)
- 시간대가 같은가 (예: 매화 낮 방문)

**구조 변화 도구:**
- 공간 전환: NURI 밖 (재윤의 집, 거리, 진서의 공간, 대중교통 안)
- 시간 전환: 밤, 새벽, 평일/주말 교차
- 시점 이벤트: 위기/사고/블랙아웃/예고 없는 수신으로 시작
- 인물 조합: 2인 장면, 단독 장면, 평소 안 만나는 조합
- 정보 방향: 재윤이 묻는 대신 추론하는 화, 아름이 주도하는 화, 이석호가 통제력을 잃는 화
- 종결 변화: 클리프행어, 타인 시점 에필로그, 시간 점프, 질문으로 끝남

**매 화 집필 전 자문:** "이번 화를 직전 화와 구별하는 구조적 차이가 최소 2개 이상 있는가?"

## 절대 금지

- 물리적 역행 (타임머신으로 몸이 과거로 가는 것)
- 도약 없이 과거를 아는 것처럼 묘사
- 하린의 정체를 아크 4 이전에 직접 언급
- 이석호가 자신의 상태를 스스로 "도약 손상"이라고 말함 (몰라야 함)
- 이석호를 단순 악역으로 묘사 (그의 논리는 일관성이 있음)
- 이석호의 혼탁 상태와 명료 상태를 같은 장면에서 혼용 (한 장면당 하나의 상태 유지)
- 어트랙터를 "마법처럼" 이탈 (근본 원인 제거만으로는 불가. REWIND 교란 중단이 필요)
- 이석호의 도약 횟수(47회)를 반전 공개 전까지 본문에 노출 (서술자 내레이션, 재윤 내면, 대사 모두 금지)
- 재윤 시점 본문에서 "명료/혼탁 상태"라는 용어 사용 금지 (재윤은 이석호가 가끔 다른 사람 같다는 감각만 있음. 분류명을 모름)

## 주제의식 집필 원칙

이 이야기의 두 축:

1. **숭고함** — 미래를 모르는 채 걷는 인간의 아름다움
2. **저주** — 미래를 알게 된 존재가 잃어버리는 것들

집필 시 반드시 적용:

- 도약 이전의 재윤은 따뜻하게, 도약 이후는 조금씩 건조하게 묘사
- 아름이 등장할 때: 불확실성과 현재에 사는 사람임을 표현 (저주의 반대)
- 하린의 대화: "이미 알고 있지만 실행한다"는 피로감이 배어야 함
- 도약 횟수가 쌓일수록 재윤의 내면 문체가 계산적으로 변해야 함
- 평범한 일상 장면(도약 없는 시간)은 아끼며 써라 — 그것이 이 이야기에서 가장 귀한 시간이다

**"알아버림으로써 잃은 것"이 보이지 않으면 이 이야기가 아니다.**

## SF 집필 특수 규칙

- 세계관 이론을 "설명"하지 말고 "사건으로 보여줘라"
- 과학 개념은 캐릭터의 감정을 통해 전달
- 패러독스는 논리로 해설하되, 독자가 따라갈 수 있는 속도로
- 매 아크마다 복선 RL 코드 2개 이상 심기
